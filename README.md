# Welcome to webpack-for-react-vue-template

## 模版下载
``` bash
$ git clone https://github.com/xulayen/webpack-for-react-template.git && cd webpack-for-react-template
```

## 模版安装

``` bash
$ npm i
```

## 开发

- 开发过程中使用代理访问服务器

``` bash
$ npm run start

$ npm run dev_server_win
```

## 打包

``` bash
$ npm run build
```

## 启动后端服务

``` bash
$ npm run server_win

$ npm run server_linux

$ npm run server_mac
```

## 打包之后访问

http://127.0.0.1:8011/

    ...

    router

    static
